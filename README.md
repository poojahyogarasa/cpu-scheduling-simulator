An interactive **CPU Scheduling Algorithm Simulator** that visualizes how different scheduling algorithms work in an operating system.

The simulator allows users to:

- Add processes dynamically
- Visualize process execution using **Gantt Charts**
- Calculate scheduling **performance metrics**
- Compare multiple algorithms
- Automatically determine the **best scheduling algorithm**

# 📊 Supported Scheduling Algorithms

The simulator implements the following algorithms:

1. **First Come First Serve (FCFS)**
2. **Shortest Job First (SJF)**
3. **Shortest Remaining Time First (SRTF)**
4. **Round Robin (RR)**
5. **Priority Scheduling**

---

# ⚙️ Features

### ✔ Process Management
- Add new processes
- Edit existing processes
- Delete processes
- Automatic PID generation

### ✔ Dynamic Inputs
- Arrival Time
- Burst Time
- Priority (for Priority Scheduling)
- Quantum Time (for Round Robin)

### ✔ Visualization
- Interactive **Gantt Chart**
- Process execution timeline

### ✔ Performance Metrics
The simulator calculates:

- Average Waiting Time (WT)
- Average Turnaround Time (TAT)
- Average Response Time (RT)

### ✔ Algorithm Comparison
The system compares all algorithms and automatically identifies the **best algorithm** based on performance.

### ✔ Graphical Analysis
Charts are generated for:

- Average Waiting Time
- Average Turnaround Time
- Average Response Time
